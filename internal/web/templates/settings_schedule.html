Schedule

Automated schedule and manual runs

Nightly Schedule

--:--

Reclaimer runs automatically every night at this time. It processes all enabled rules, evaluates criteria, and takes the configured action on matching items. Changing this takes effect on the next scheduled cycle.

User & Watch History Sync

Refreshes users and watch history straight from Plex and Jellyfin. Runs independently from rule scans.

How often to refresh user watch history. Default: every 6 hours.

Manual Run

Trigger a run right now without waiting for the nightly schedule. A dry run shows you what would happen without actually making any changes -- useful for testing new rules. A full run will take real actions (unmonitor, delete, etc.) based on your rule settings.
